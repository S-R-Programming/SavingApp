# 節約履歴アプリ
このアプリは「日々の節約」を具体的な金額で記録するAndroidアプリです。

## 注意点
これは就職活動のポートフォリオとして、作ったアプリです。

## 開発環境 
Android Studio (Java)

## 使い方

## 工夫点
・現在、自分が持っている知識を使いました。<br>・Firebaseを用いたアカウント作成、ログイン機能、データベース管理<br>・Twitterでツイート出来るようにしました
<br>・デフォルトのFirebaseのログイン機能がなぜか上手くいかなかったので自分で実装しました。
